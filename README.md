# نظام التسويق الميداني — بوابة الإدخال + الداشبورد

## محتوى المشروع
- `index.html` — نموذج إدخال البيانات (بوابة التسويق)
- `dashboard.html` — داشبورد التقارير
- `Code.gs` — كود الربط مع Google Sheets (Google Apps Script)

كلا الملفين مرتبطين بنفس الشيت عبر نفس رابط الـ Web App، وتم فحصهما والتأكد إن:
- ترتيب الأعمدة في `Code.gs` مطابق تمامًا لترتيب الحقول في كل نموذج (تمويل / مسئول أول / مسئول جغرافي).
- رابط الـ Web App موجود ومتطابق في الملفين.
- تمت إضافة حماية (HTML escaping) في الداشبورد لمنع أي مشاكل لو دخلت بيانات تحتوي رموز خاصة.

---

## الخطوة 1 — تجهيز الشيت (لو لسه ما عملتهاش)
1. أنشئ Google Sheet جديد.
2. أضف 4 تابات بنفس الأسماء المذكورة في تعليقات أعلى `Code.gs`:
   `الموظفين` ، `تمويل` ، `مسئول اول` ، `مسئول جغرافي`
3. من القائمة: `Extensions > Apps Script`، احذف أي كود موجود، والصق محتوى `Code.gs` بالكامل.
4. اضغط `Deploy > New deployment`.
5. النوع: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
6. اضغط Deploy وانسخ رابط **Web app URL**.

> لو غيّرت النشر لاحقًا وطلع رابط جديد، لازم تحدّث المتغير `APPS_SCRIPT_URL` في **الملفين** (`index.html` و `dashboard.html`).

---

## الخطوة 2 — رفع المشروع على GitHub
1. من GitHub اعمل **New repository** (مثلاً باسم `marketing-portal`).
2. ارفع الملفات الثلاثة: `index.html`، `dashboard.html`، `Code.gs` (والـ README ده لو حبيت).
3. من تبويب **Settings > Pages**:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
   - احفظ.
4. GitHub هيديك رابط شبيه بـ:
   `https://USERNAME.github.io/marketing-portal/`

الروابط النهائية هتكون:
- بوابة الإدخال: `https://USERNAME.github.io/marketing-portal/`
- الداشبورد: `https://USERNAME.github.io/marketing-portal/dashboard.html`

بما إن GitHub Pages بيشتغل تلقائيًا على **HTTPS**، خدمة تحديد الموقع (GPS) في نموذج الإدخال هتشتغل مباشرة من غير أي إعداد إضافي.

---

## ملاحظات مهمة
- ضيف كل موظف وفئته في تاب "الموظفين" قبل ما يقدر يسجل دخول بالأوراكل بتاعه.
- الداشبورد بيتحدّث بالضغط على زرار "تحديث" في أعلى الصفحة، ويجيب أحدث بيانات من الشيت مباشرة.
- لو ظهرت رسالة "تعذر الاتصال بالنظام" تأكد إن رابط الـ Web App لسه فعّال (لم يتم حذف الـ deployment).
